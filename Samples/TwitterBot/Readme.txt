
Overview
========

This sample interacts with the Twitter API to provide Twitter services to users
on an IRC network.

Command-Line Commands
=====================

Exit the program
----------------

Syntax:

  exit

Connect to a server
-------------------

Syntax:

  connect <server address>

Examples:

  connect irc.freenode.net
  connect 192.168.2.3

Disconnect from a server
------------------------

Syntax:

  disconnect <server name regex>

Examples:

  disconnect freenode

Join a channel
--------------

Syntax:

  join <server name regex> <channel name>

Examples:

  join freenode #ircsil
  
Leave a channel
---------------

Syntax:

  leave <server name regex> <channel name>

Examples:

  leave freenode #ircsil
